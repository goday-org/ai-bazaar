# AI Bazaar

> 去中心化 AI 订阅二级市场的协议与参考实现。

**当前状态**：pre-alpha，交接给 Codex 开发中。

---

## 这是什么

让有富余 AI 订阅额度（Claude / OpenAI / Gemini / Antigravity）的人，通过密封竞价匿名转售给需要的人。完全依靠 GitHub 去中心化运作，端到端加密，链上托管资金。
